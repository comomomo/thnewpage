Concrete Dream Engine - 2015

Concrete Dream Engine is a looping realtime application which presents of a series of Brutalist styled buildings in the idealized language of architectural rendering. Buildings in the work often reference actual counterparts, many of which have been demolished.

I developed this work by modelling structures first in 3dsMAX and rendering them in Unity. The complete work runs infinitely in a linear loop through each shot presented in the video documentation. Only cloud structures vary.
