Hug Point, Oregon 45.831015, -123.962879 - 2015

A processing sketch uses SimpleOpenNI libraries to render kinect points in space. Points are placed in a scene composed of LIDAR scan data of Hug Point (along the Oregon Coast).

Built off of Max Rheiner's example in the SimpleOpenNI libary: "SimpleOpenNI AlternativeViewpoint3d Test" and tai_pinc's code for loading points from xyz point clouds. Oregon LIDAR available through PSU's public collection.
