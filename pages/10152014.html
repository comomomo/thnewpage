Block vs Inline

Images by default are inline

Center with style: 

option 1

     display: block;
     width: 100px;
     margin: 0 auto;

option 2

     position: absolute
     top: 100px;
     left: 50%;
     margin-left: -50px; (accounts for size of the image and offset center)

bottom is also cool

css property

               -moz-animation: dogmove 1s infinite;
